# 🎵 AudioConvert — Free Audio Converter

A full-stack audio conversion platform built with **React + Vite** (frontend) and **Flask + PostgreSQL** (backend).

## Features
- **Format Conversion**: Convert between WAV, MP3, FLAC, AAC, OGG, M4A, AIFF, ALAC.
- **Quality Control**: Custom bitrate and bit depth selection.
- **Drag & Drop**: Easy file upload interface.
- **Account System**: Freemium model with usage limits and automatic file cleanup.

## Tech Stack
- **Frontend**: React, Vite, CSS
- **Backend**: Python, Flask, FFmpeg, SQLite/PostgreSQL
- **Payments**: Stripe Integrations
